ExGen 시작하기


* ExGen은 Ext JS를 기반으로한 프레임워크 이므로 Ext JS에 대한 사전 기술이 필요합니다.
* ExGen을 사용하기 위해서 ExGen 홈페이지의 ExGen 시작하기 메뉴의 내용을 참조하시기 바랍니다.

1. 애플리케이션 시작위치

로그인 -> 공통코드가져오기 -> 메뉴가져오기 -> 메인화면
시작페이지 경로는 다음과 같습니다.
view / main / Main.js -> view / login / Login.js



Main.js를 더블클릭하면 화면이 나타납니다. 이때 해당 코드를 보고싶다면 상단의 코드보기를 클릭하면 됩니다.





2. 로그인 처리

로그인 화면은 기본적으로 다음과 같은 화면을 제공합니다. 화면과 데이터는 수정하여 사용하실 수 있습니다.



로그인은 다음 경로에 위치합니다.

view / login / Login*.js



로그인을 하게 되면 다음과 같은 호출과정을 거칩니다.

view / login / Login*.js -> controller / AppController.js (loginAfter) -> 공통코드호출



로그인 정보는 서버에서 내려주어야 하며 ExGen CLOUD 에서는 Json파일로 대신합니다.
JSON 파일은 수정할 수 있으며 아래 경로에서 제공합니다.

json / login / login.json



로그인된 후 속성은 다음 Js 파일의 객체에 저장됩니다.

resources / js / common.js




exCommon.user = { userId:'', userName:'', addr:'', tel:'', brcd:'', brcdName:'' };


3. 공통코드

공통코드는 controller / AppController.js 에서 로그인 후 자동 호출되며 별도의 화면은 없습니다.

controller / AppController.js (getCommonCode)




공통코드는 서버에서 내려주어야 하나 ExGen CLOUD 에서는 Json파일로 대신합니다.
JSON파일은 수정할 수 있으며 아래 경로에서 제공합니다.

json / common / common.json




호출된 공통코드는 store / CommonStore.js 에 저장됩니다.



저당 후 메뉴를 호출합니다.


4. 메뉴가져오기

메뉴는 controller / AppController.js 에서 로그인-> 공통코드 호출 후 자동 호출됩니다.

controller / AppController.js (getMenu) - 메인화면이 텝화면일 경우
controller / AppController.js (getDesktopMenu) - 메인화면이 텝화면일 경우




메뉴는 사용자별로 다를 수 있으므로 서버에서 내려주어야 하나 ExGen CLOUD에서는 Json 파일로 대신합니다.
JSON 파일은 수정할 수 있으며 아래 경로에서 제공합니다.

json / menu / menuTab.json
json / menu / menuDesktop.json




JSON파일은 2개가 있으며 menuTab.json은 화면에 탭화면일 경우이며 menuDesktop.json은 메뉴화면인 웹 데스크탑형태일 경우 사용합니다.

탭화면일 경우 메뉴정보는 다음 js 파일의 객체에 저장됩니다. js파일은 자신의 메뉴구조에 맞게 수정하여 사용합니다. (테스트탑은 Json 파일을 바로 메뉴에 적용합니다.)

json / common / common.json




exCommon.menu = { topMenu1:{}, topMenu2:{}, topMenu3:{}, topMenu4:{}, topMenu5:{} };
모든 정보를 가져오면 전체 화면인 view / main / Main.js 는 Login.js 를 제거하고 다음 파일을 로딩합니다.
view / main / MainPanel.js (탭패널일 경우)
view / main / MainDesktop.js (웹 데스크탑일 경우)





5. 메인화면선택

메인 화면을 탭화면 또는 데스트탑 화면으로 설정하려면 controller / AppController.js 파일에서 아래 속성을 변경하여 처리할 수 있습니다.

mainScreenCls:'tabpanel', // tabpanel, desktop





6. 폴더생성

메인 설정이 모두 끝났으며로 화면별 개발을 진행합니다.
먼저 메뉴에 맞게 폴더를 생성합니다.
>

참고 - 버튼기능)
a) 툴바의 제일 왼쪽 버튼은 폴더를 생성합니다.
b) 두번째 버튼은 파일을 생성합니다.
c) 세번째 버튼은 파일이나 폴더명을 변경합니다.
d) 네번째 버튼은 파일이나 폴더를 삭제합니다.
e) 다섯번째 버튼은 파일을 업로드합니다.
f) 여섯번째 버튼은 파일을 다운로드합니다.
g) 마지막 버튼은 리스트를 갱신합니다.


7. 화면 만들기

상단의 화면만들기 버튼을 클릭하여 업무별 화면을 만듭니다.



화면만들기는 홈페이지의 데스크탑 사용설명서 -> 화면만들기 메뉴 그룹을 참조하시기 바랍니다.


8. 실제 화면보기

모든 개발이 완료되었으면 실제 화면을 확인해 볼 수 있습니다.



설정한 로그인, 공통코드, 메뉴를 이용해 최종 화면을 보여준다.


(별도로 변경하지 않아도 기본적으로 설정된 상태로 화면이 나타납니다.)
